El directorio 'Python code' contiene dos archivos en formato python, en donde se encuentra el código básico utilizado.
Para ejecutar los archivos pyton se deben descargar e instalar las librerías utilizadas en el trabajo.

El directorio 'CleanData' es un proyecto del IDE NetBeans con el lenguaje java. Este software se lo utilizó para: formatear, ordenar y reemplazar datos, mediante archivos csv.



En el archivo 'Diccionario.txt' se detalla las caracteristicas y su codificacion utilizada para la ejecucion del presente proyecto.

El archivo 'R-004-SDSW-UTI-UNL-2020-Reporte-Historico-Matriculados-Sistemas-Computacion.xlsx' contiene la data original, emitida desde UTI-UNL.

El archivo 'Data preporcesada.csv' contiene la data preprocesada a la que se la utilizó con los algoritmos.

El archivo 'Arbol.dot' es el arbol generado mediante el lenguaje python y para la visualizacion gráfica se lo debe abrir desde una herramienta para formatos '.dot'.

El archivo 'datain2.csv' contiene unicamente los números de cedula de los registros estudiados. Este valor se lo utiliza para trasformar y ordenar los datos mediante el proyecto 'CleanData' de java.
